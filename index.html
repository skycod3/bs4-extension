Introdução

Essa é uma extensão baseada no framework Bootstrap v4 que tem como objetivo adicionar funcionalidades importantes para a empresa e seguir com a filosofia do framework: fornecer código mais padronizado, dinâmico e com mais produtividade.

Para entender essa documentação, primeiro é preciso entender a documentação oficial do Bootstrap. Não é responsabilidade dessa documentação explicar o que já está esclarecido na documentação oficial. A documentação pode ser encontrada acessando o link: https://getbootstrap.com/docs/4.4/getting-started/introduction/

Nem sempre foi possível preservar alguns dos padrões do framework Bootstrap porque algumas partes dessa extensão teve que se adequar à realidade da empresa.

Essa é uma documentação ainda em desenvolvimento, funcionalidades podem se adicionadas, alteradas ou removidas.

Padrões de Media Queries

As classes dessa extensão que suportam media queries seguem o mesmo padrão de nomenclatura do framework Bootstrap.

Essa extensão segue suportando os mesmos breakpoints: sm, md, lg, xl.

O padrão de nomenclatura é o seguinte: {propriedade}-{breakpoint}-{valor}.

Alterações

Abaixo estão relacionadas as mudanças da versão 1.0.0 para a mais recente e estável 2.0.0.
A versão 2.0 é incompatível com versões anteriores, caso queira utilizar a versão recente em projetos antigos, bugs graves irão ocorrer.

Tanto a versão antiga como a mais recente podem ser encontradas por meio do link https://github.com/agencianetrs/bs4-extension

Background

Com a finalidade de reduzir o código, algumas propriedades da versão anterior foram alteradas.

v1.0.0: bg-position-left-top, bg-repeat-yes, bg-size-cover, bg-attachment-fixed

v2.0.0: bg-left-top, bg-repeat, bg-cover, bg-fixed

Somente o valor inherit teve a propriedade total conservada. (bg-position-inherit, bg-repeat-inherit, bg-size-inherit, bg-attachment-inherit).

Border

Como no caso anterior a propriedade border-width foi alterada.

Os valores em pixels foram ajustados para representar a medida real.

v1.0.0: border-width-14 (30px)

v2.0.0: border-30 (30px)

Offset

Os valores em pixels foram ajustados para representar a medida real.

Os valores em porcentagem permaneceram iguais.

Com a finalidade de deixar o código mais claro para iniciantes, as propriedades foram alteradas.

v1.0.0: ot-0, or-5, ob-10, ol-n12

v2.0.0: top-0, right-5, bottom-30, left-n50

Sizing

Foram acrescentados valores em pixels para: width, height, max-width e min-height

Os valores em porcentagem permaneceram iguais.

v1.0.0: sem valores em px

v2.0.0: w-30, h-150, maxw-700, minh-600

Spacing

Os valores em pixels foram ajustados para representar a medida real.

v1.0.0: mt-6, p-10

v2.0.0: mt-10, p-30
